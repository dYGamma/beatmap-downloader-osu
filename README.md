# osu! Beatmap Downloader

Десктопное приложение для массового скачивания карт из вкладки «Больше всего сыграно» любого osu! профиля. Написал потому что у самого был трабл с потерей мап, а качать каждую вручную это хуня собачья.

## Что умеет

- Авторизация через osu! OAuth (браузер открывается сам, пароль нигде не хранится)
- Поиск любого профиля по нику или ID
- Скачивает полный список «Больше всего сыграно», даже если там 3000+ карт
- Сохраняет всё как `.osz` файлы в папку по выбору
- Пропускает карты которые уже есть
- 5 потоков загрузки параллельно, не ждёшь вечно
- Работает через зеркала (catboy.best, osu.direct, chimu.moe), osu! Supporter не нужен

## Настройка

Приложение работает через официальное osu! API, так что нужно один раз создать OAuth-приложение. Минут 5 от силы:

1. Заходишь на [osu.ppy.sh/home/account/edit](https://osu.ppy.sh/home/account/edit), листаешь до раздела **OAuth**
2. Нажимаешь **New OAuth Application**
3. В Callback URL вставляешь: `http://localhost:8080/callback`
4. Сохраняешь, получаешь Client ID и Client Secret
5. Вставляешь их в приложение при первом запуске

Всё, больше это делать не придётся.

## Как пользоваться

1. Запускаешь приложение, входишь в osu! аккаунт
2. Вводишь ник или ID профиля (например `10874153`), жмёшь **Найти**
3. Выбираешь папку куда сохранять
4. Жмёшь **Начать загрузку** и ждёшь

В процессе видно что качается, что пропускается и что не скачалось. По завершении папку можно открыть прямо из приложения и перекинуть всё в Songs.

## Как работает скачивание

Официальный osu! требует Supporter для загрузки, поэтому используются публичные зеркала:

1. **catboy.best** (основное)
2. **osu.direct** (запасное)
3. **chimu.moe** (последний вариант)

Если карта недоступна на всех трёх, скорее всего она удалена с osu!. Загрузка идёт в 5 потоков с небольшой задержкой чтобы зеркала не банили. На 3000 карт примерно 30-60 минут в зависимости от интернета.
